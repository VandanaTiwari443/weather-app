# Weather App 🌤️

This is a simple Weather Application built using HTML, CSS, and JavaScript.

## Features
- Check current weather by city
- Search any city weather
- World cities weather
- Temperature, humidity, wind, pressure details

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## Pages
- Home Page
- Search Weather
- World Weather

## How to Use
1. Open the website
2. Search your city name
3. View weather information
